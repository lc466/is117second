# Resume instructions

Place a PDF resume at `resume/resume.pdf`. This file is linked from `index.html` as the downloadable resume.

Recommended workflow:

1. Keep a source resume (Markdown or Google Docs).
2. Export to PDF and save as `resume/resume.pdf`.
3. Commit and push to GitHub — GitHub Pages will serve the file.

If you want automated PDF exports from Markdown, consider using Pandoc locally.
